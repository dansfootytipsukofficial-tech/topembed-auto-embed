Step 1: Your GitHub Pages URL

After setting up this repository, your streams will be hosted at:

https://[your-username].github.io/[repo-name]/embed/

Example: https://dansfootytipsukofficial-tech.github.io/topembed-auto-embed/embed/

Step 2: Embed Code for Weebly

In your Weebly editor, add an "Embed Code" element and paste:

<div style="width:100%;max-width:1200px;margin:0 auto;">
  <iframe
    src="https://[your-username].github.io/[repo-name]/embed/"
    width="100%"
    height="900"
    style="border:0;display:block;"
    allowfullscreen
    title="Live Sports Streams">
  </iframe>
</div>

How It Works

1
Daily Automation
GitHub Actions runs every day at 6 AM UTC, fetching the latest sports streams from TopEmbed API.
2
Stream Extraction
Puppeteer extracts working HLS stream URLs from the TopEmbed channels.
3
HTML Generation
Python script generates a clean, mobile-responsive player page with all available channels.
4
GitHub Pages Hosting
The generated site is published to the gh-pages branch and served via GitHub Pages (free hosting).
5
Weebly Embed
You embed the GitHub Pages URL in your Weebly site using an iframe. Your visitors see fresh streams every day!
